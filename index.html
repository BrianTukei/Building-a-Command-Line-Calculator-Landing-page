Command Line Calculator

A simple calculator application accessible from the command line.

Try out the calculator for yourself: Deployed App

Feature 1: Basic Arithmetic

The Command Line Calculator supports basic arithmetic operations like addition, subtraction, multiplication, and division.

Feature 2: Advanced Functions

The calculator also includes advanced functions like square root, exponentiation, and logarithms.

Feature 3: Error Handling

The calculator is designed to handle errors gracefully, providing helpful feedback for invalid inputs.

About

Hi! I'm Tukei Brian, a software developer with a passion for building innovative solutions.

I created this command-line calculator as a personal project to demonstrate my skills
in programming and problem-solving.

The Command Line Calculator was inspired by my own need for a quick and easy way
to perform calculations from the terminal. As a software engineering student at Holberton School, I wanted to apply my skills
to a practical problem and create a useful tool for myself and others. This project was
completed over the course of several weeks in June 2024.
